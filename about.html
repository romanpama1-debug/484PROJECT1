About My Topic

This is to help do Project 1

This website was created as a practice project to demonstrate the basics of HTML structure, navigation, and CSS styling. The goal of this project is to show how a simple layout can grow into a full website by adding new pages, images, and interactive features over time. While building this site, I learned how to organize content using semantic HTML elements, create consistent page layouts, and style content using CSS. This project represents the first step toward building more advanced and responsive websites in the future.
